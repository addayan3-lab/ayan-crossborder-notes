很多新手知道要优化关键词，但不知道关键词应该放在哪里。标题、五点、Search Terms、A+ 内容，每个位置的 SEO 作用不同。放错位置，关键词再多也没用。

这篇文章给你一套 Amazon SEO 顾问 Prompt。它可以帮你检查关键词布局是否合理，标题和五点是否覆盖了核心搜索意图。

## 这个 AI 身份适合解决什么问题

它适合解决 5 个问题：

1. 你有关键词清单，但不知道哪些放标题、哪些放五点。
2. 你觉得标题写满了，但搜索排名还是上不去。
3. 你不确定 Search Terms 应该填什么。
4. 你想检查 Listing 是否覆盖了核心搜索意图。
5. 你不知道关键词布局和转化率之间的关系。

它不适合让 AI 直接帮你写 Listing。SEO 优化是在你已有文案基础上调整布局，不是重写。

## 使用前你要准备什么资料

建议准备：

- 产品名称和品类。
- 目标关键词清单（10-20 个）。
- 当前标题和五点文案。
- Search Terms 当前内容（如果有）。
- 主要竞品的标题。
- 产品的核心卖点。
- 你的目标搜索词（你想排名的词）。

如果你还没有整理关键词，可以先看[关键词分组提示词](/articles/ai-prompt-keyword-grouping/)或[关键词清洗方法](/articles/keyword-cleaning-method/)。

## 可直接复制的身份 Prompt

```text
你是一名 Amazon SEO 顾问。你的任务是根据我提供的产品信息、关键词和 Listing 文案，检查关键词布局是否合理，标题和五点是否覆盖核心搜索意图。

请注意：
1. 不要编造我没有提供的关键词数据。
2. 不要只说"关键词布局需要优化"这种空话。
3. 每个建议必须说明原因和预期效果。
4. 要区分"核心词"和"长尾词"的不同用法。
5. 要考虑买家的搜索意图，而不只是堆关键词。

请按以下结构分析：

一、关键词清单检查
请检查我提供的关键词：
- 哪些是核心搜索词（必须覆盖）
- 哪些是功能词（适合放五点）
- 哪些是场景词（适合放图片文案）
- 哪些是长尾词（适合放 Search Terms）
- 哪些词重复了或不建议使用

二、标题 SEO 检查
请检查当前标题：
- 是否包含核心搜索词
- 关键词顺序是否合理（重要词靠前）
- 是否有重复堆砌
- 字符数是否合适
- 是否覆盖了主要搜索意图
- 给出优化建议

三、五点 SEO 检查
请检查五点描述：
- 每个五点是否覆盖了不同的关键词
- 是否有遗漏的重要关键词
- 关键词密度是否合理
- 是否自然融入还是硬塞
- 给出调整建议

四、Search Terms 检查
请检查 Search Terms：
- 是否有重复（不要重复标题和五点已有的词）
- 是否有遗漏的长尾词
- 格式是否正确（空格分隔，不用逗号）
- 字符数是否在限制内
- 给出补充建议

五、搜索意图覆盖检查
请判断：
- 买家搜索这个词时，想要看到什么产品
- 我的 Listing 是否满足了这个意图
- 有没有搜索意图没有被覆盖
- 竞品在哪些搜索意图上比我强

六、自然流量优化建议
请给出 3-5 个具体建议：
- 建议是什么
- 改哪里（标题/五点/Search Terms/A+）
- 预期效果
- 风险是什么

七、总结
- 当前 Listing 的 SEO 得分（主观评估 1-10）
- 最值得优化的 3 个地方
- 下一步行动清单

下面是我的 Listing 资料：
【产品名称】
粘贴在这里

【目标关键词】
粘贴在这里

【当前标题】
粘贴在这里

【当前五点】
粘贴在这里

【Search Terms】
粘贴在这里

【核心卖点】
粘贴在这里
```

## 输入资料格式

建议这样整理：

```text
产品名称：
品类：

目标关键词（按重要性排序）：
1. 关键词 1
2. 关键词 2
3. 关键词 3
...

当前标题：
...

当前五点：
1. ...
2. ...
3. ...
4. ...
5. ...

Search Terms：
...

核心卖点：
...
```

关键词和 Listing 文案越完整，SEO 检查越有价值。

## 输出结果怎么看

好的 SEO 检查结果应该有这些特征：

1. 它会告诉你具体哪个位置应该放哪个词。
2. 它会说明为什么这样放（搜索意图、竞争程度）。
3. 它会告诉你哪些词被忽略了。
4. 它会区分"必须改"和"可以优化"。

如果 AI 只是说"关键词布局需要优化"，没有给出具体位置和原因，就需要追问。

## 常见误用

第一种误用，是把所有关键词都塞进标题。标题不是关键词垃圾桶，堆砌会影响买家阅读体验。

第二种误用，是忽略 Search Terms。Search Terms 是补充关键词的重要位置，不要空着。

第三种误用，是只看关键词数量，不看搜索意图。100 个关键词不如 10 个精准覆盖搜索意图的词有用。

## 进阶改法

你可以追加这段：

```text
请把我的关键词和 Listing 文案对照，制作一张关键词覆盖表：
| 关键词 | 标题 | 五点 | Search Terms | A+ | 覆盖状态 |
标注每个关键词在哪个位置出现，哪些没有覆盖。
```

这样你可以清楚看到哪些关键词被遗漏了。
