很多新手卖家写 Listing 的时候容易"写过头"。材质明明是不锈钢，写成"医用级不锈钢"；效果只是防水，写成"完全防水"；明明没有认证，写成"通过 FCC 认证"。这些表达一旦被平台发现或被买家投诉，轻则警告，重则下架。

这篇文章给你一套提示词，让 AI 帮你检查 Listing 里可能存在的风险表达。注意：AI 不是合规最终判断，最终要以亚马逊官方政策为准。

## 这个提示词解决什么问题

它主要解决 4 个问题：

- 新手不知道哪些词在亚马逊上是违规词。
- Listing 写完后没有人检查，直接上架了。
- AI 写的内容没经过核验，包含虚假信息。
- 绝对化用语、未证明的效果、未授权认证是最常见的三个风险区域。

## 使用前你要准备什么

你需要准备：

- 完整 Listing 文案：包括标题、五点描述、产品描述或 A+ 文案。
- 产品实际信息：材质、功能、认证、规格等。
- 你确认能提供的证明：哪些功能你有测试报告，哪些认证你确实通过了。

如果你还没写五点描述，可以先看[让 AI 帮你写五点描述的提示词](/articles/ai-prompt-bullet-points-writing/)，写完之后再回来检查。

## 可以直接复制的提示词

```text
你是一名亚马逊 Listing 合规检查顾问。你的任务是帮我检查 Listing 文案中可能违反平台政策或存在风险的表达。

请注意：
- 你不能只检查明显的违规词，还要检查暗示性表述。
- 你要区分"绝对违规"和"证据不足可能违规"。
- 你要基于我提供的产品实际信息做判断，不能凭空猜测。
- 你的检查不能替代亚马逊官方政策，最终判断以平台规则为准。

请根据我提供的 Listing 文案和产品实际信息，完成以下检查：

一、夸大风险
找出所有可能夸大的表达：
1. 使用了"最""第一""顶级""完美"等绝对化用语。
2. 夸大了产品效果或性能范围。
3. 暗示了不能保证的使用结果。
4. 修改建议（如何改写以避免风险）。

二、未证明的效果
找出所有缺少证据支撑的表述：
1. 宣称了某种效果但没有测试报告支持。
2. 使用了"医用级""食品级""专业级"等需要认证的表述。
3. 宣称了比较结果（如"比某某更好"）。
4. 修改建议（如何改写或补充证据）。

三、绝对化表达
找出所有可能被认定为虚假宣传的表述：
1. "永不""绝不会""完全"等绝对化用词。
2. "适合所有"类型等不可能证实的表述。
3. "保证退款""保证满意"等承诺性表述。
4. 修改建议。

四、可能违规词
找出亚马逊严格限制或禁止的词汇：
1. 品牌名称比附（如"兼容 iPhone"需授权）。
2. 专利/版权相关表述（如"专利设计"需真实）。
3. 医疗/健康功效类表述（如"治疗""预防"）。
4. 环保类表述（如"可降解""环保材料"需认证）。
5. 修改建议。

五、需要人工确认的地方
1. 需要查看检测报告才能确认的内容。
2. 需要查看认证文件才能使用的表述。
3. 需要参考最新平台政策的内容。
4. 建议咨询平台客服或专业人士的问题。

输出格式：
- 按风险等级分类输出。
- 每项风险标注：原文位置、风险类型、修改建议、紧急程度。
- 最后输出需要人工确认的重点问题清单。

禁止事项：
- 不要把我提供的信息当成"可替换"建议。
- 不要把不确定的内容归为"绝对违规"。
- 不要给出"这样改一定安全"的承诺。
- 不要忽略平台政策可能随时更新。

下面是 Listing 文案和产品信息：
【Listing 文案】
标题：
五点描述：
产品描述 / A+：

【产品实际信息】
材质：
功能：
认证情况：
可提供证明：
不能夸大的地方：
```

## 输入资料格式

建议你这样整理输入：

```
Listing 文案：
标题：家用不锈钢保温杯，24 小时保温，医用级 316 材质，适合所有人群
五点：
1. 医用级 316 不锈钢材质，安全无毒
2. 24 小时超强保温，永不降温
3. 完美密封设计，100% 不漏水
4. 适合户外、办公、居家各种场景
5. 最好的保温杯，销量第一

产品实际信息：
材质：304 不锈钢，不是 316
功能：保温实测 12 小时
认证：无
可提供证明：材质报告（SGS）
不能夸大的地方：不能保证保温 24 小时，没有医用级认证，没有销量第一数据
```

## AI 输出后怎么判断好坏

好的风险检查应该具备以下特征：

- 明确区分了"绝对违规"和"证据不足"。
- 每项风险都给出了可修改的具体建议。
- 主动标出了需要人工确认的地方。
- 修改建议不依赖你没有的信息。

如果 AI 只输出"请遵守平台规则""建议如实描述"这种通用提示，说明检查不够深入。

## 常见误用

**第一种误用**，是以为 AI 检查过就没有风险了。AI 不是亚马逊内部审核系统，它的判断有滞后性，平台规则每年都会更新。

**第二种误用**，是让 AI 检查自己不知道的信息。如果你给 AI 的 Listing 里本身就有你编造的内容，AI 不一定能全部识别。

**第三种误用**，是修改完不复查。AI 建议你把"最好"改成"优质"，但你改完后整句话可能依然有问题。

## 进阶改法

如果你检查的是某个特定类目，可以在提示词里追加：

```text
请特别注意以下类目的常见违规风险：厨房类目、母婴类目、医疗健康类目。并说明每条风险对应的亚马逊政策条款编号（如果已知）。对于不确定的政策条款，标注"建议查阅最新官方政策"。
```

不同类目的违规风险差异很大，母婴和食品类目比亚马逊工具类目敏感得多。
