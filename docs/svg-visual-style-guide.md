# SVG 视觉风格规范

## 统一视觉风格

- **氛围**：商务教育感，适合亚马逊卖家运营培训场景
- **布局**：卡片式信息图，清晰分区，模块化
- **背景**：浅色（#F8FAFC / #F0F4FF），配合柔和渐变
- **主色**：#2563EB（蓝色）
- **辅助色**：#D97706（琥珀色，用于标注/提醒）、#059669（绿色，用于完成/正向指标）、#F43F5E（红色，用于风险/警告）
- **质感**：圆角矩形卡片、轻微阴影、透明装饰圆
- **标题**：加粗、大号、深色（#0F172A），副标题灰蓝色（#475569）

## 禁止元素

- Amazon 官方 logo 和商标
- 亚马逊后台截图
- 真实订单数据、金额、账号信息
- 暴富暗示、跑车、现金雨
- 夸大收益承诺（"月入百万"、"保证爆单"）
- 外部图片引用（`<image href="..."`）
- 外部字体（仅使用系统字体 `system-ui, -apple-system, sans-serif`）
- JavaScript 或动画

## SVG 尺寸规范

- **宽度**：1200px
- **高度**：630px
- **viewBox**：`"0 0 1200 630"`
- 不使用外部资源
- 不使用远程图片
- 文件名为 `cover.svg`

## Alt 文案规范

- 使用中文
- 准确描述图片用途
- 不堆砌关键词
- 不使用夸张营销用语
- 格式：`"<主题>示意图"` 或 `"<主题>封面图"`
- 示例：`"亚马逊关键词学习路径封面图"`

## 文件命名规范

- 全英文小写
- 使用 `cover.svg`
- 不允许空格
- 不允许中文文件名
- 目录名对应文章 slug 或功能模块
